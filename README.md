### Hi, I'm Cesare

**Nuclear Engineering student at [Politecnico di Milano](https://www.polimi.it/)**, building open-source tools at the intersection of engineering, computation, and data.

I turn domain knowledge into software that solves real problems — from satellite collision avoidance to industrial energy optimization to automated statistical discovery.

---

#### Featured Projects

| Project | Description | Highlights |
|---------|-------------|------------|
| **[SatGuard](https://github.com/cesabici-bit/satguard)** | Satellite conjunction assessment pipeline | TLE ingest → SGP4 → collision probability (Foster/Chan/Alfano) → 3D CesiumJS globe → maneuver planning. Validated against Vallado & NASA CARA |
| **[HeatScout](https://github.com/cesabici-bit/heatscout)** | Industrial waste heat recovery screening tool | 8 recovery technologies, CAPEX/NPV/IRR estimates, Sankey diagrams, PDF reports. 168 tests, validated against real plant data. [**Live demo**](https://heatscout.streamlit.app) |
| **[OmniOracle](https://github.com/cesabici-bit/omni-oracle)** | Automatic statistical truth discovery engine | Ingests 500+ public time series, discovers lagged relationships via mutual information + Granger causality with FDR correction |
